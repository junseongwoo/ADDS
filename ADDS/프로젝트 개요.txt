ADDS = Automated Defect Detection System
- 자동 결함 검출 시스템 개발 

## 프로젝트 주요 기능 ## 
1. 실시간 이미지 캡처 및 처리
2. 이미지 전처리 (노이즈 제거, 이미지 정규화)
3. 결함 검출 알고리즘 구현
4. 검출 결과 시각화 및 리포트 생성
5. 설정 가능한 검사 파라미터 관리

## 확장 기능 ## 
1. 다양한 종류의 결함 분류
2. 실시간 모니터링 대시보드
3. 데이터베이스 연동

## 프로젝트 만들면서 고려해야하는 부분 ## 
1. 객체지향 설계 원칙
    - 설정 분리
    - 모듈화
    - 확장 가능한 구조

2. 이미지 처리 기본 기술
    - 노이즈 제거
    - 이미지 정규화
    - 이진화
    - 윤곽선 검출

3. 성능 최적화
    - 실시간 처리를 위한 최적화
    - 메모리 관리

4. 산업용 소프트웨어 개발 실무
    - 로깅
    - 리포트 생성
    - 설정 관리
